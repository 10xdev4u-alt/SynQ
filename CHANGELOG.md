# Changelog

All notable changes to the CommitsLedger project will be documented in this file.

## [Unreleased] - 2026-01-04

### Added
- Command-line options support (--dry-run, --verbose, --config, --log, --help)
- Configuration file support with default settings
- Dry-run mode to preview operations without making changes
- Logging capabilities with timestamped entries
- Enhanced error handling and validation
- Connectivity checks before attempting git operations
- Improved progress indicators with commit messages
- Security enhancements and input validation
- Detailed documentation in README

### Changed
- Removed informal language and emojis for professional appearance
- Refactored code into modular functions for better maintainability
- Improved error messages and user feedback
- Enhanced script validation and safety checks
- Updated usage examples and configuration documentation

### Fixed
- Better handling of authentication failures
- More robust branch detection and remote validation
- Improved error recovery during push operations
- Fixed issues with non-existent remote branches
